SYSTEM DESCRIPTION
===========================================
BAGO.PH is a web-based prototype of a mobile waste management
platform that connects Filipino barangay residents, garbage
collectors, and LGU officials in one real-time system.

The system addresses the communication breakdown in Philippine
barangay waste management by providing real-time collection
schedules, photo-based issue reporting, a gamified eco-points
rewards system, QR code segregation verification, and an LGU
analytics dashboard.

===========================================
HOW TO RUN
===========================================
1. Open index.html in any modern web browser (Chrome recommended)
2. Navigate using the top navigation bar
3. To view XML transformed by XSL, open schedules.xml directly
   in a browser that supports XSLT (Firefox recommended for XML)
4. Import bago_ph_database.sql into phpMyAdmin or MySQL Workbench
   to set up the database schema and sample data
